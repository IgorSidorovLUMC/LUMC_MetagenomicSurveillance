# Supplementary data
Code and Data Repository
This repository contains scripts and supplementary data associated with the study:
Mourik et al. – “Viral metagenomic sentinel surveillance of acute respiratory infections in primary care as a public health strategy: a prospective national pilot study.”

The materials provided here enable transparent, reproducible, and FAIR (Findable, Accessible, Interoperable, and Reusable) analyses of viral metagenomic sequencing data, supporting public health utility of metagenomic approaches in respiratory infection surveillance.
